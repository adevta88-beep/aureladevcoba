# Study Planner Online

## Latar Belakang
Siswa sering kesulitan mengatur tugas dan jadwal belajar. Website ini dibuat untuk membantu mengorganisir tugas, deadline, dan prioritas dengan mudah.

## Manfaat
- Mengatur tugas lebih mudah
- Mengurangi stres
- Membantu manajemen waktu
- Memberi saran otomatis dengan bantuan AI

## Cara Menggunakan
1. Buka halaman website
2. Masukkan nama tugas, deadline, dan prioritas
3. Klik "Tambah Tugas"
4. Lihat daftar tugas yang muncul
5. Gunakan fitur Sort untuk mengurutkan
6. Gunakan fitur AI untuk saran otomatis
